## Usage

1. **Create Notes**
   - Click the **`+` button** to open color options and add a new note.
   - Select a color from presets, use the random color generator, or type a custom hex code.

2. **Edit Notes**
   - Click inside any noteâ€™s textarea to edit the content.

3. **Delete Notes**
   - Click the **trash icon** to remove a note permanently.

4. **Favourite Notes**
   - Click the **star icon** to mark/unmark a note as favourite.

5. **Filter Notes**
   - Use the filter buttons (All, Favourite) to sort notes by category.

6. **Search Notes**
   - Type keywords in the search box to instantly find matching notes.

7. **Multi-Color Panel**
   - Open the panel via the **multi-color trigger**, pick or mix colors, and create colorful notes.

8. **Notifications**
   - Every action shows a small toast alert (e.g., â€œNote created successfully!â€).

9. **Default Notes**
   - On first visit, five default notes appear with tips and colorful welcome messages.
   - Some default notes can even be pre-marked as favourites for quick access.

10. **FAQ Section**
    - Toggle FAQ questions for quick guidance within the app.
